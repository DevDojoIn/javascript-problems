Problem 2:Write a js program to display area and perimeter of a rectangle.

Solution:


    function calculateRectangle(length, width) {
      // Calculate area and perimeter
      const area = length * width;
      const perimeter = 2 * (length + width);
      
      // Display the results
      console.log("Area: " + area + " square units");
      console.log("Perimeter: " + perimeter + " units");
      }
      
      // Example usage
      calculateRectangle(10, 5);

Code Breakdown:

  • calculateRectangle(length, width) is a function that takes two parameters: length and width.
  • The Area of the rectangle is calculated using the formula:
    area = length * width.
  • The Perimeter of the rectangle is calculated using the formula:
    perimeter = 2 * (length + width).
  • The function prints the calculated area and perimeter using console.log().
  • The function is called with values 10 and 5, and the result is printed in the console.
