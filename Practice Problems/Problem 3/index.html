Problem 3: Write a program to find the total surface area and volume of a cuboid.

Solution:


    function calculateCuboid(length, breadth, height) {
        const totalSurface = 2 * (length * breadth + breadth * height + length * height);
        const volume = length * breadth * height;
        return `The TSA of cuboid is ${totalSurface} and the volume of cuboid is ${volume}.`;
    }
    console.log(calculateCuboid(5, 10, 6));

Code Breakdown:

  • calculateCuboid(length, breadth, height) is a function that takes three parameters: length, breadth, and height.
  • The Total Surface Area (TSA) of a cuboid is calculated using the formula:
    2 * (length * breadth + breadth * height + length * height).
  • The Volume of the cuboid is calculated using the formula:
    length * breadth * height.
  • The function returns a formatted string containing both the TSA and the volume.
  • The function is called with values 5, 10, 6, and the result is printed using console.log().
